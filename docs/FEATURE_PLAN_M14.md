# Feature Plan — M14: Менеджмент чатов

## Цель

Родитель-администратор может видеть список всех чатов (общий + приватные) и очищать сообщения в любом из них.

---

## Требования

### Список чатов

- Отдельный пункт навигации в хедере для `parent`: **Чаты**
- Отображается как таблица со столбцами:
  - Название (общий чат — "Общий", приватные — по именам участников, например "Иван + Мария")
  - Участники
  - Количество сообщений
  - Дата последнего сообщения
  - Кнопка "Очистить"
- Показываются все чаты: общий + все приватные (включая чаты с Lulu)
- Доступно только для `parent`

### Очистка сообщений

- Кнопка "Очистить" удаляет все сообщения в чате
- Перед очисткой — диалог подтверждения ("Вы уверены?")
- После очистки — чат остаётся, пользователи видят пустую ленту
- Удаляются только сообщения, сам чат не удаляется

### Доступ

- Только `parent` видит эту страницу и может выполнять очистку
- `child` не имеет доступа

---

## Out of scope

- Удаление самого чата
- Просмотр содержимого чата из этой страницы
- Очистка отдельных сообщений (только весь чат целиком)
- Доступ к менеджменту чатов для `child`
