# Feature Plan — M11: UI Redesign

## Цель

Переработать визуальный стиль приложения: игровой дизайн, тёмная тема, компонентная библиотека shadcn/ui, шрифт Nunito. Целевая аудитория — дети от 11 лет.

---

## Дизайн-решения

- **UI библиотека:** shadcn/ui (поверх Tailwind CSS)
- **Тема:** тёмная, игровая — насыщенные акцентные цвета (фиолетовый / синий / неоновые детали)
- **Шрифт:** Nunito (Google Fonts) — округлый, дружелюбный, читабельный
- **Стиль:** карточки с закруглёнными углами, градиенты, мягкие тени, крупные кнопки

---

## Что нужно сделать

### M11.1 — Установка и настройка shadcn/ui + Tailwind

- Установить Tailwind CSS в проект frontend
- Установить shadcn/ui
- Настроить тёмную тему (CSS переменные в `globals.css`)
- Подключить шрифт Nunito через Google Fonts в `index.html`
- Настроить `tailwind.config.js`: цвета, шрифт Nunito как `font-sans`

### M11.2 — Редизайн экрана логина (`LoginPage`)

- Центрированная карточка на тёмном фоне с градиентом
- Логотип / название приложения с игровым стилем
- Поля ввода в стиле shadcn/ui (`Input`)
- Кнопка входа (`Button`) — крупная, акцентного цвета
- Сообщения об ошибках — стилизованные

### M11.3 — Редизайн хедера (`AppHeader`)

- Тёмный хедер с градиентом или акцентной полосой
- Логотип/название слева
- Навигация (для parent): стилизованные `NavLink` с подсветкой активного
- Справа: имя пользователя + кнопка Logout в стиле shadcn/ui

### M11.4 — Редизайн чата (`ChatPage`)

- Тёмный фон страницы
- Лента сообщений:
  - Сообщения в пузырях (`bubble`) с закруглёнными углами
  - Свои сообщения — справа, акцентный цвет
  - Чужие сообщения — слева, нейтральный тёмный цвет
  - Сообщения бота Lulu — особый стиль (иконка + цвет)
  - Имя отправителя и время — мелко, над/под пузырём
- Форма ввода:
  - Тёмное поле ввода, стилизованное
  - Кнопка отправки — иконка или текст, акцентный цвет
  - Счётчик символов
- Статус соединения — стилизованная полоска/бейдж

### M11.5 — Редизайн управления пользователями (`UserManagementPage`)

- Список пользователей в виде карточек или таблицы shadcn/ui (`Table`)
- Кнопки действий (создать, сменить пароль, деактивировать) — стилизованные
- Модальное окно или форма создания пользователя — shadcn/ui `Dialog`
- Статус пользователя (активен/деактивирован) — цветной бейдж

## Затронутые файлы

### Новые
- `frontend/tailwind.config.js`
- `frontend/src/globals.css` (или `index.css` — глобальные CSS переменные темы)
- `frontend/components.json` (shadcn/ui конфиг)
- `frontend/src/components/ui/` — shadcn/ui компоненты (Button, Input, Card, Dialog, Table, Badge...)

### Изменяемые
- `frontend/index.html` — подключить Nunito из Google Fonts
- `frontend/src/pages/LoginPage.jsx` — редизайн
- `frontend/src/pages/LoginPage.module.css` — заменить на Tailwind классы
- `frontend/src/components/AppHeader.jsx` — редизайн
- `frontend/src/components/AppHeader.module.css` — заменить на Tailwind классы
- `frontend/src/pages/ChatPage.jsx` — редизайн
- `frontend/src/pages/ChatPage.module.css` — заменить на Tailwind классы
- `frontend/src/pages/UserManagementPage.jsx` — редизайн
- `frontend/src/pages/UserManagementPage.module.css` — заменить на Tailwind классы
- `frontend/package.json` — новые зависимости
