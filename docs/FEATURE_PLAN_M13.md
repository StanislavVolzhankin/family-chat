# Feature Plan — M13: Приватные чаты

## Цель

Пользователи могут создавать приватные чаты друг с другом. Переписка видна только участникам чата — другие пользователи, включая родителя, доступа не имеют.

---

## Требования

### Создание чата

- Любой пользователь (parent и child) может создать приватный чат с любым другим пользователем
- Правый клик на имени в списке онлайн-пользователей → кнопка "Создать приватный чат"
- Чат создаётся мгновенно, без подтверждения второго участника
- Один чат на пару пользователей — повторное создание открывает существующий чат

### Lulu в приватном чате

- Lulu отображается в списке онлайн-пользователей как обычный участник
- Можно создать приватный чат только с Lulu (правый клик → "Создать приватный чат")
- В приватном чате только с Lulu бот отвечает на любое сообщение (без `@Lulu`)
- Lulu можно добавить в существующий приватный чат между двумя людьми — кнопка внутри окна чата
- Lulu можно удалить из чата — кнопка внутри окна чата
- При добавлении Lulu в чат с двумя людьми — `@Lulu` обязателен для получения ответа
- История сообщений Lulu остаётся после её удаления из чата

### Доступ и приватность

- Видят переписку только участники чата
- Родитель не имеет привилегированного доступа — он такой же участник
- Приватность обеспечивается на уровне API: 403 для посторонних

### UI

- Если приватный чат существует — иконка появляется рядом с именем пользователя в списке онлайн
- Иконка одна, независимо от того, есть Lulu в чате или нет
- Клик на иконку → чат открывается во всплывающем окне
- Иконка отображается только пока пользователь онлайн (нет пользователя онлайн — нет иконки)
- Счётчика непрочитанных нет
- "Закрыть" = закрыть всплывающее окно; чат и история сохраняются

### Правила сообщений

- Те же ограничения: 150 символов, антиспам 1 сообщение/сек
- Retention 30 дней
- Сообщения доставляются в реальном времени через WebSocket (как в общем чате)

### Деактивация участника

- Деактивированный пользователь принудительно выходит из системы
- Чат и история остаются

---

## Out of scope

- Счётчик непрочитанных сообщений
- Удаление чата и истории
- Добавление других пользователей (не Lulu) третьим участником
- Уведомления о новых сообщениях
