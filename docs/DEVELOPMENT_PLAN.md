# Family Chat — Development Plan

## Milestone 0 — Scaffold
**Цель:** пустой, но рабочий проект с правильной структурой.
- Laravel-приложение с модульной структурой (Auth, Chat)
- React SPA с роутингом и заготовками страниц
- PostgreSQL migrations (таблицы users, messages)
- Docker-конфигурация для backend
- `.env.example` файлы

## Milestone 1 — Аутентификация
**Цель:** пользователь может войти в систему.
- `POST /api/auth/login` → JWT токен
- Экран логина (React)
- Хранение токена, редирект на чат
- Обработка ошибок (неверный логин/пароль)
- Локализация строк (RU/EN) для экрана логина

## Milestone 2 — Управление пользователями
**Цель:** родитель может создавать аккаунты детей.
- `POST /api/users` — создание ребёнка
- `GET /api/users` — список пользователей
- `PATCH /api/users/{id}` — смена пароля / деактивация
- UI для родителя (отдельная страница)
- Защита роутов: дети не видят этот раздел

## Milestone 3 — Чат (базовый)
**Цель:** пользователи могут обмениваться сообщениями в реальном времени.
- `GET /api/messages` — история за 30 дней
- WebSocket-сервер (подключение с токеном)
- Отправка `send_message`, получение `new_message`
- UI чата: лента сообщений + форма ввода
- Антиспам (1 сообщение/сек)
- Лимит 150 символов

## Milestone 4 — Устойчивость и ошибки
**Цель:** система корректно ведёт себя при проблемах.
- Reconnect с exponential backoff
- UI-состояния: онлайн / переподключение / недоступен
- Блокировка отправки при потере соединения
- Обработка ошибок БД и сервиса
- Retention-job (удаление сообщений старше 30 дней)

## Milestone 5 — Чат-бот Lulu
**Цель:** пользователи могут задавать вопросы боту Lulu прямо в чате.
- Обращение через `@Lulu` в общем чате
- Backend: Bot модуль (BotService, интеграция с OpenAI API)
- Миграция: поле `is_bot` в таблице `users`, запись Lulu в БД (seeder)
- Валидация: нельзя создать пользователя с именем бота
- Ответ бота в общем чате, хранится в БД как обычное сообщение
- Обработка ошибок OpenAI (локализованное сообщение)
- Frontend: иконка бота у сообщений Lulu
- Имя бота настраивается через `BOT_NAME` в `.env`

## Milestone 7 — Навигационное меню администратора
**Цель:** родитель видит навигационное меню в хедере для переключения между разделами.
- Top-меню только для `parent`: Чат, Менеджмент пользователей
- Активный пункт подсвечивается
- После логина — редирект на Чат
- В хедере справа: имя пользователя + Logout
- `child` меню не видит

## Milestone 8 — Список активных пользователей
**Цель:** все пользователи видят кто сейчас онлайн в сайдбаре чата.
- Сайдбар справа от ленты сообщений
- Presence Channel (Laravel Reverb) для отслеживания подключений
- Имя + зелёный индикатор для каждого онлайн-пользователя
- Lulu всегда в списке с иконкой бота

## Milestone 9 — Chat Improvements
**Цель:** улучшить UX чата и устранить задержку доставки сообщений.
- Отправка сообщения по `Enter` (`Shift+Enter` — перенос строки)
- Устранение задержки доставки 3–5 сек (диагностика + фикс queue/broadcast)

## Milestone 10 — Деплой
**Цель:** проект живёт в интернете.
- Docker + `fly.toml` для Fly.io
- Vercel конфиг для React
- GitHub Actions CI/CD pipeline
- Переменные окружения и секреты
